# Voice-Versa

## Project Setup Directions

1. Clone the repository
2. Create a virtual environment
  `python3 -m venv venv`
3. Activate the virtual environment
   Windows: `./venv/Scripts/Activate.ps1`
4. Install dependencies
  `pip install -r requirements.txt`
5. Run Application
  `flask run`
6. Enjoy!
